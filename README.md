# Instructions
Electric and software project for navigation of the mini-sumo 500g

Developed by Gustavo Rocha Serafim in 2019

# Components

Arduino UNO

Driver motor L298N

3x Sensor E18-D80NK

Battery 7v

2x motors DC 6v with current stall 2A

2x Sensor Line QRE

Protoboard

Jumpers
